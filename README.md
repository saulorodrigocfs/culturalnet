# 🌎 CulturalNet

Uma plataforma web para compartilhar conteúdos culturais com a comunidade.

Os usuários podem publicar posts com título, categoria, conteúdo e imagem, criando um feed colaborativo de informações culturais.

---

## 📸 Preview

Feed de publicações com imagens e categorias, semelhante a uma timeline simples de rede social.

*(Em breve)*

---

## 🚀 Tecnologias Utilizadas

- ASP.NET Core MVC
- C#
- Entity Framework Core
- SQL Server
- Razor Pages
- Bootstrap
- HTML / CSS

---

## ✨ Funcionalidades

✔ Criar posts culturais  
✔ Adicionar categoria ao post  
✔ Publicar conteúdo textual  
✔ Upload de imagem para o post  
✔ Feed organizado por data (mais recente primeiro)  
✔ Armazenamento de imagens no servidor  

---

## ⚙️ Como Rodar o Projeto

1️⃣ Clone o repositório

git clone https://github.com/saulorodrigocfs/culturalnet.git

2️⃣ Entre na pasta do projeto

cd CulturalNet

3️⃣ Execute as migrations

dotnet ef migrations add 
dotnet ef database update

4️⃣ Rode o projeto

dotnet run

A aplicação abrirá no navegador.
